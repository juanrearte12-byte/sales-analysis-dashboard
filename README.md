# 📊 Sales Analysis Dashboard (Excel)

## 🧠 Descripción
Este proyecto consiste en el análisis de un dataset de ventas obtenido de Kaggle, utilizando Excel para explorar, visualizar y obtener insights relevantes del negocio.

Se desarrolló un dashboard interactivo que permite analizar el rendimiento de ventas desde distintas perspectivas.

---

## 🛠️ Herramientas utilizadas
- Microsoft Excel
- Tablas dinámicas
- Gráficos dinámicos
- Segmentación de datos (Slicers)

---

## 📊 KPIs principales
- Total de ventas
- Unidades vendidas
- Ticket promedio

---

## 📈 Análisis realizados
- Ventas por región
- Ventas por categoría de producto
- Ranking de vendedores
- Ventas por método de pago

---

## 🔍 Funcionalidades
- Dashboard interactivo mediante filtros (slicers)
- Visualización clara y estructurada
- Análisis exploratorio de datos (EDA)

---

## 💡 Insights obtenidos
- Las ventas se concentran en determinadas regiones
- Algunas categorías dominan el ingreso total
- Existe una diferencia significativa entre vendedores
- El método de pago influye en el volumen de ventas
